Eu sou Hugo Bessa, desenvolvedor front-end e entusiasta de colaboração. Atualmente trabalho na Mkt Virtual e moro em Santos, São Paulo.

Eu tenho um interesse: conectar pessoas e ideias. Eu acredito que é possível mudar o mundo quando esses dois ingredientes explosivos se juntam. Construir pontes entre os dois é o que escolhi fazer. Este site é mais uma tentativa de levar ideias interessantes para pessoas incríveis.

Eu também acredito que a melhor forma de aprender é fazendo. E a melhor forma de fazer é colaborativamente. Então, me envolver com projetos Open Source foi um caminho natural. Eu estou no GitHub como @hugobessaa.

Você pode entrar em contato comigo pelo email eu@hugobessa.com.br. Ou por um tweet pra @hugobessaa.

Cólofon

Este blog é todo um grande experimento. Aqui eu aproveito para brincar e validar diversas ideias. Seja uma nova ideia para produção de conteúdo, método de desenvolvimento ou maneira de encarar design.

A seguir, algumas das tecnologias que tenho usado por aqui:

Sublime Text 3

A maioria de todas as linhas de código que já escrevi desde que entrei no mundo front-end foram tecladas dentro deste maravilhoso editor. Durante todo esse tempo aprendi muitos atalhos úteis, plugins sensacionais e formas de dar ao código toda a atenção que ele merece.

Jekyll

Em vez de usar uma arquitetura back-end ultra-complicada e sofisticada, eu resolvi simplificar as coisas e usar um gerador de site estático. Isso facilita na parte do servidor mas também facilita na hora de espremer cada byte deste blog para deixá-lo o mais leve possível.

O Jekyll também permite que eu use Markdown, uma incrível abstração de HTML básico de formatação, para escrever meus posts.

Scss

Desde que comecei a usar um pré-processador para CSS, nunca mais voltei atrás. Uma ferramenta essencial, o Scss me ajuda a maneter meus códigos de estilo bem divididos e reutilizáveis. Aqui eu junto mais de 30 arquivos em um só CSS com ~5KB.

Github

Ferramenta indispensável para mim. É lá que guardo todos os meus projetos de código aberto, incluindo este site, que está disponível abertamente por lá.

O Git por si só também é uma ferramenta essencial dentro de cada um dos meus projetos.

Grunt

Eu odeio perder tempo. E o Grunt me ajuda a aproveitar mais cada momento que estou desenvolvendo. Ele me ajuda a compilar stylesheets, otimizar imagens, minificar recursos e a publicar cada alteração por aqui. Eu até já apresentei uma palestra sobre ele: Automação de Tarefas com Grunt.
